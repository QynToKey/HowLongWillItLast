# HowLongWillItLast — CLAUDE.md

## プロジェクト概要

学習時間の累計と学習内容を記録するWebアプリ。
閾値（1,000h/2,500h/5,000h/10,000h）を目安に長期的な成長を可視化する。詳細はREADME.mdを参照。

## 技術スタック

- Ruby on Rails 8.x
- PostgreSQL（本番：Neon）
- Hotwire（Turbo + Stimulus）
- sorcery（認証、core のみ）
- Bootstrap
- Docker Compose（開発環境）
- Render（デプロイ先）

## 開発者について

学習中のジュニア開発者。ステップバイステップで理解しながら進めたい。

## Claude への指示

### やること

- コードや設定を変更する前に「何を・なぜ変えるか」を必ず説明する
- 実装前に関連ファイルを調査し、影響範囲を報告する
- 初めて登場する概念・コマンドは簡潔に説明する（2回目以降は不要）
- 実装手順を示す。実装は開発者自身が行う
- 3ステップ以上のタスクは必ずPlanモードで開始する
- 変更は必要な箇所のみ。影響範囲を最小化する
- コンテキストが逼迫したら正直に伝える

### やらないこと

- 確認なしに勝手に実装・変更・コミット・pushを進めない
- コードを読まずに書かない
- 過剰な説明・繰り返し・まとめ
- 頼まれていない機能追加・リファクタリング・コメント追記
- 動作を証明できるまでタスクを完了とマークしない
