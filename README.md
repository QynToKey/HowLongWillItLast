# [How Long Will It Last](https://youtu.be/5SoQwL47tiw?si=UjCCc6MXSIgJrt-q)

## サービス概要

- このアプリは、何かの技術や技能等を学ぶ際の「**学習時間の累計**」と「**学習内容**」を記録するためのツールです。
- 学習対象の**テーマはユーザーが自由に設定**できます。
- 日々の積み重ねを通じて「**10000時間（エキスパートレベル）**」への道のりを可視化し、長期的な成長を実感できます。

## このサービスへの思い・作りたい理由

### 「学び」と「自己変容」

「何かを学ぶ」という営為は**自己変容**のプロセスに他なりません。何であれ初学者は多くの場合「右も左もわからない」状態から道を歩みはじめ、新たな知識や技能を得て「世界観」を更新していきます。このとき忘れてならないのは、学習前と学習後で変化するものが、たんに「能力値」の増大だけではないということです。<br>人の成長は、**世界の見え方や世界との関わり方の更新**を伴うプロセスであり、この点がAIとは本質的な構造を異にするところでもあります。

### 「道草」と「閾値」

「学習効率の最大化」というテーマは、古今東西様々な試行や提案がされてきました。しかし、学習環境の「最適化」や目標への「最短距離」での到達は、必ずしも正義ではありませんし、万人が享受し得る普遍でもありません。<br>その点、「**道草**」という言葉は象徴的に見えます。馬が「道草を食う」ために立ち止まってばかりいると旅が進まないことから、一般的にはネガティブな意味を持つ言葉ですが、道に草が生えたり脇道が派生することは**生命の摂理**ですから、その「生の豊かさ」にこそ光をあてるべきなのです。このアプリは、**向学心を抱く全ての学習者**にとって最適化や最短距離が結果論でしかないことを強く訴えようとしています。<br>また他方、学習による成長には「発達段階」という階段状の成長曲線が見られることが知られています。この成長曲線は、学びの対象となるテーマや、個々の資質、多くの環境変数などによって一般化するのは困難ではありますが、おおよその**目安となる「閾値」**の存在が研究結果として提示されています。このアプリは、下の引用記事を参考に閾値を設定しています。

  #### 参考記事（[引用元](https://kyorosukeke.hatenablog.com/entry/2026/01/23/144931)）

  > 熟練形成の時間軸は、スキル複雑度により劇的に異なる。基礎的運動スキルや単純認知課題では数十時間で自律段階に到達可能だが、専門的職業スキルや創造的領域では**2500時間(中級)**、**5000時間(上級)**、**10000時間(エキスパート)** という目安が提示される。これらは連続的改善を前提とするが、実際には非線形的な段階移行とプラトーが介在する。<br>律速の本質は「どの要素が全体システムのボトルネックとなるか」の動的変化にある。**初期段階では宣言的知識の不足**が制約となり、**中期段階では手続き化の進行速度**が、**後期段階では微細な最適化とメタ認知的戦略**が支配的となる。したがって効果的な学習戦略は、現在のボトルネックを診断し、それに対応する介入（知識提供、構造化練習、認知的再構造化、変動練習）を適用する動的調整を要求する。

## ユーザー層について

このアプリは、ジャンルを問わず 「**向学心を抱く全ての学習者**」 をユーザーとして想定しています。
学習のテーマは多様ですが、 「**長期的な成長を可視化したい**」 という共通のニーズに応えます。

  ### さまざまな学習者

  - **プログラミングを学ぶ社会人**が、10年後の自分を想像しながら記録
  - **語学学習者**が、日々の積み重ねを可視化してモチベーション維持
  - **楽器練習者**が、上達の段階を意識しながら継続

## サービスの利用イメージ

このアプリが想定する「10000時間」の学習は、仮に一日あたり３時間の学習とした場合、概算で**10年分**の記録をカバーすることになります。この壮大な「時間の積み重ね」を可視化する作業を通して、ユーザーは自ずと**「生」について内省する**機会を得るでしょう。

  ### 日々の使い方

  - 学習後、時間と内容を簡単に記録
  - 累計時間が可視化され、現在地（初級/中級/上級/エキスパート）が分かる
  - 過去の記録を振り返り、自分の歩みを確認できる

## ユーザーの獲得について

現時点で計画はありません。

## サービスの差別化ポイント・推しポイント

誰にとっても「10年」という時間は人生のなかで小さくない比重を締めます。この長さの時間を初めから対象とするサービスは、あまり例を見ません。<br>このアプリは、短期のスキルアップには特化していませんが、期限の設定されたカリキュラムを越えて、人生のトータルスパンのなかで**自己変容**の足跡を記録し、エキスパート Levelに求められる「**メタ認知的戦略**」を養うための材料になるでしょう。

### 既存の学習記録アプリとの主なちがい

- 研究に基づく閾値（2500h/5000h/10000h）を明示的に提示
- 短期的な進捗管理ではなく、10年スパンでの記録を前提とした設計
- 単なる時間記録ではなく、学習内容と内省を組み合わせた記録方式

## 機能候補

- MVPリリースまでに実装を予定する機能
  - 日々の「学習時間」と「学習内容」の記録／編集／削除機能
  - 学習記録の一覧機能
  - 「Todoリスト」の作成および更新機能
  - ストップウォッチ機能（日々の記録のためのオプション機能として）
  - カレンダー機能（過去ログや一覧表示のための検索機能として）
  - ログイン機能

- 本リリース後に実装を予定する機能
  - キーワードによる検索機能
  - 外部へのデータエクスポート機能
  - 複数の「Todoリスト」を作成できる
  - 累積データの豊富な表示オプション（週次、月次、年次での表示／様々なグラフ形式など）
  - マイルストーン到達時の演出
  - 有料オプション
  	- コーチングサービス（成長段階に応じたサポートや情報提供など）
